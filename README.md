# The Filecoin Name Service
    Our goal is to establish a fully open-source, complete, sustainable operation, 
    everyone can participate in and use and gain benefits, 
    and a name system on FILECOIN that is fully compatible with ENS. We call this system FNS!

    We hope that FNS can prosper together with FILECOIN, 
    promote the growth of decentralized facilities together, 
    and become one of FILECOIN's infrastructures!

    The TOKEN we issue is also named FNS

## FNS volume and distribution:
    The main difference between FNS and ENS lies in the economic model: FNS tokens are minted entirely by users when they apply for a domain name

    FNS tokens are completely minted by users when they apply for a domain name. The total circulation is 100,000,000 pieces, and the development team gets 10% of them.

    The first person who pays FIL to apply for a domain name can get 90 FNS, 
    and the nth person who pays FIL to apply for a domain name will mint (1 + ((99 * (1980198 — n)) / 1980198)) FNS,
    The domain name application address gets 9/10, and the development team gets 1/10; when n >= 1980198, 
    the domain name application cannot get FNS,
    Using FNS to apply for a domain name will not mint and obtain FNS, nor will it affect the count of n

## FNS function:
    FNS can be used to apply for and refresh the expiration time of domain names
    When applying for and updating the expiration time of domain names, 
    1 FNS is equivalent to 1$. For example, if you apply for fildomains.fil with a one-year validity period, 5 NFS is required

    Of course, fil can also be used to apply for and refresh the expiration time of domain names
    For example, if you apply for fildomains.fil with a one-year validity period, you can pay the equivalent of $5 fil

## FNS gain:
    The development team gets 10% of the proceeds from applying for and refreshing the domain expiration date, 
    and the remaining 90% goes into a token pool called Sunday

    The Sunday token pool only allows transfers, pledges and withdrawals of pledges on Sundays, 
    and income can be received at other times

    1/64 of the current remaining FIL and FNS of the Sunday token pool are shared by all pledge holders each week

    As in the Sunday tokens pool pledged a total of 100 FNS, address 0 10 FNS x5fbdb2315678afecb367f032d93f642f64180aa3 pledge,
    Sunday token pool balance is 6400 FNS, 640 FIL,
    Is this week 0 x5fbdb2315678afecb367f032d93f642f64180aa3 can get 10 FNS and 1 FIL earnings

## FNS 总量和铸造：
    FNS和ENS 主要不同在于经济模型： FNS代币完全由用户在申请域名时铸造

    FNS的代币完全由用户在申请域名时铸造，总发行量100,000,000枚，开发团队获得其中的10%

    第一个支付FIL申请域名的可以获得90个FNS, 第n个支付FIL申请域名会铸造(1 + ((99 * (1980198 — n)) / 1980198)) 个FNS, 
    域名申请地址获得其中9/10, 开发团队获得1/10； 当n >= 1980198时，申请域名不能获得FNS，
    使用FNS申请域名不会铸造和获得FNS，也不会影响n的计数

## FNS 作用：
    FNS 可以用于申请和刷新域名到期时间，在申请和更新域名到期时间时1 FNS相当与1$, 如以一年有效期申请 fildomains.fil，需要 5 NFS
    
    当然申请和刷新域名到期时间也可以用fil支付, 比如以一年有效期申请 fildomains.fil，可以支付相当于 5$ 的fil

## FNS 收益：
    申请和刷新域名到期时间的收益的10% 由开发团队获得， 剩余的90% 会存入一个叫Sunday 的代币池
    
    Sunday 代币池只在星期天才允许转账, 质押和收回质押，其他时候可以领取收益
    
    每个星期由所有质押者分享 Sunday 代币池当前剩余的FIL和FNS的1/64
    
    如在 Sunday 代币池总共质押 100 FNS, 地址 0x5FbDB2315678afecb367f032d93F642f64180aa3 质押了 10 FNS, 
    Sunday 代币池的余额为 6400 FNS, 640 FIL, 
    则本周 0x5FbDB2315678afecb367f032d93F642f64180aa3 可以领取 10 FNS 和 1 FIL 的收益
